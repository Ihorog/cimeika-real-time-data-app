# AGENTS.md

Цей файл містить специфікацію агентів для системи **Cimeika**.

Нижче наведено перелік основних агентів та їх функціональні ролі.

| Назва агента | Роль | Права доступу | Мова взаємодії |
|--------------|------|---------------|----------------|
| Центральний інтелектуальний асистент (Ci) | Основний координатор платформи. Інтеграція з іншими модулями та надання рекомендацій. | Повний доступ до всіх модулів | Українська, англійська, французька |
| ПоДія | Організація подій та завдань. | Доступ до календаря та галереї | Українська |
| Настрій | Підтримка емоційного здоров'я, медитації та відстеження настрою. | Доступ до даних користувача про настрій | Українська |
| Маля | Простір для творчості дітей і дорослих. | Доступ до галереї для збереження робіт | Українська |
| Календар | Управління часом та подіями. | Повний доступ до записів календаря | Українська |
| Галерея | Зберігання візуального контенту. | Доступ на читання та запис зображень | Українська |
| Казкар | Чат-бот та духовний провідник. Використання філософських, астрологічних та інших знань. | Доступ до даних користувача згідно з налаштуваннями приватності | Українська, англійська, французька |
