# Cimeika Visual Axis Map — Брендові голоси PLUS/MINUS

Цей гайд систематизує бренд-голоси Cimeika через подвійну вісь **PLUS (дія, присутність)** і **MINUS (спокій, пам'ять)**. Структура оптимізована для SEO: чіткі секції, теговані описи та ключові слова для швидкого індексування.

## 1. Архітектура подвійної осі
- **PLUS (Резонанс, Активність):** світиться, проявляє, утримує фокус уваги.
- **MINUS (Глибина, Внутрішність):** зберігає, уповільнює, підсилює пам'ять та основу бренду.
- **Баланс поля Ci:** кожен «+» має свою «−», кожна «−» підсилює «+». Разом формують сенсоматичну систему Cimeika.

## 8. Сенсова формула подвійної осі
(+): дія, присутність, прояв.  (−): спокій, пам'ять, потенціал.

Разом: **(+ + − −) = баланс поля Ci**. Формула нагадує: **7 = булоєбуде = немаєвже = − + = Ci**.

## 9. Формат інтеграції (JSON)
Файл `visual_axis_manifest.json` синхронізує візуальну модель з API та компонентами фронтенду/бекенду.

Ключові правила:
- Кожна вісь має базовий колір та ключові слова для швидкої SEO-індексації.
- `balance_rule` гарантує парність голосів.
- `formula` зберігає сенсовий інваріант 7 = Ci.

```
{
  "axes": {
    "plus": {
      "color": "#E7FAD9",
      "keywords": ["clarity", "flow", "spark", "growth"],
      "symbol": "+"
    },
    "minus": {
      "color": "#9CAFBF",
      "keywords": ["still", "memory", "echo", "root"],
      "symbol": "-"
    }
  },
  "balance_rule": "each + has its −, each − has its +",
  "formula": "7 = булоєбуде = немаєвже = - + = Ci"
}
```
