# Інтерфейсна система Cimeika

## Мета
"Cimeika" — інтерактивна екосистема з семи модулів, що працюють під керуванням ядра **Ci**. Ця сторінка описує єдині правила побудови інтерфейсу, адаптивності та API-взаємодій для всіх модулів.

## Архітектура модулів
| № | Модуль | Роль | Тип інтерфейсу | API |
|---|--------|------|----------------|-----|
| 1 | Ci | Ядро, логіка, контроль | Панель керування, мапа зв’язків | `/ci/chat`, `/ci/data`, `/ci/components` |
| 2 | ПоДія | Майбутні сценарії та виконання | Планер / timeline | `/api/calendar/events` |
| 3 | Настрій | Поточний емоційний стан | Візуальний шар хвиль, карти настроїв | `/ci/data` |
| 4 | Маля | Творчість, генерація | Полотно генерації, збір ідей | `/api/gallery/upload`, `ci_power` |
| 5 | Казкар | Пам’ять і архів історій | Таймлайн історій | `/ci/data` |
| 6 | Календар | Атлас часу, вузли | Карта часу та подій | `/api/calendar/events` |
| 7 | Галерея | Фотоархів | Слайдер/галерея з категоріями | `/api/gallery/upload`, `ci_power` |

## Основна палітра
- **Ci Sky `#63B3F5`** — фони, хвилі, легкість.
- **Ci Wave `#32B4F2`** — акценти, інтерактивні елементи.
- **Ci Deep `#07324F`** — структура, текст, темні секції.
- **Ci Glow `#FFD978`** та **Ci SoftGlow `#FFE8B3`** — лише як світлові ореоли та підсвітки, не як фонові заливки.
- **Ci Grass `#8BC34A`**, **Ci Calm `#4DB6AC`**, **Ci Rose `#F8BBD0`** — творчість, вода, м’якість.
- **Ci Shadow `#021726`**, **Ci Black `#020612`** — темний фон та нічний режим.

## Адаптивність
| Категорія | Діапазон | Поведінка |
|-----------|----------|-----------|
| Desktop XL | ≥1920px | Центровані блоки, простір «повітря», анімації та граф зв’язків. |
| Laptop / Tablet | 1024–1919px | Дві колонки, спрощені діаграми, стислий текст. |
| Mobile | ≤1023px | Вертикальні стеки, акордеон/таби, свайпи для дій. |

Габарити масштабуються через `clamp()` для заголовків, кнопок, іконок та полів. Горизонтальний скрол не допускається; на слабких пристроях фонова анімація вимикається.

## Навігація
- **Desktop:** верхнє меню або інтерактивна мапа вузлів Ci.
- **Mobile:** нижня іконкова панель для всіх 7 модулів, підтримка свайпів між екранами.

## Патерни компонентів
- **Карти зв’язків (Ci, Календар):** граф/SVG у Desktop, статичний прев’ю на Tablet, таби вузлів на Mobile.
- **API-консоль (Ci):** повна форма запитів на Desktop; на Mobile — спрощений JSON-view та швидкі дії.
- **Списки подій (ПоДія, Календар):** сітка/timeline на Desktop, списки зі свайпами на Mobile.
- **Настрій:** анімовані хвилі на Desktop, статичні градієнти на Mobile.
- **Галерея:** горизонтальний слайдер з hover-preview на Desktop, вертикальний список з повноекранним переглядом на Mobile.

## Принципи UX
- Немає горизонтального скролу; складні схеми редукуються мінімум у 3 кроки для мобільних.
- Інтерактиви дублюються жестами (свайп, довге натискання).
- Glow/SoftGlow застосовується тільки для світлових підсвіток.
- Темний режим (Ci Shadow / Ci Black) активується для Казкар та нічних сценаріїв.

## Інтеграції та оркестрація
- **Ci** є оператором сенсів та API-викликів, координує стан модулів.
- **ПоДія/Календар** спільно використовують `/api/calendar/events` і штовхають зміни в Ci.
- **Маля/Галерея** мають двосторонній обмін через `/api/gallery/upload` та датасет `ci_power`.
- **Настрій** та **Казкар** читають історичні зрізи з `/ci/data` для побудови хвиль та таймлайнів.

## Результати
- 7 адаптивних сторінок-модулів під єдиним стилем Ci.
- SVG/Canvas-схеми для зв’язків, таймлайнів та вузлів часу.
- Експорт матеріалів у SVG, PNG, JSON (API дані) і окремі `.md` документи по кожному модулю.
