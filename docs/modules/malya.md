# Маля — творчий генератор

## Огляд
Маля відповідає за творчий потік: генерацію ідей, ескізів та взаємодію з Галереєю.

## Ключові екрани
- **Полотно генерації:** зони для текстових промптів і візуальних варіацій.
- **Панель ідей:** швидкі шаблони, історія попередніх ітерацій.
- **Кнопка «надіслати в Галерею»:** прямий зв’язок з `/api/gallery/upload`.

## Основні компоненти
- Сітка результатів з можливістю вибору фаворитів.
- Контролери стилю/кольору з акцентами Ci Grass та Ci Rose.
- Індикатор прогресу генерації з Ci Wave.

## API взаємодія
- Вивантаження результатів у Галерею через `/api/gallery/upload` та датасет `ci_power`.
- Передача метаданих до Ci для оркестрації.

## Адаптивність
- **Desktop XL:** широка сітка прев’ю + детальна панель параметрів.
- **Laptop:** дві колонки (прев’ю + параметри).
- **Mobile:** вертикальні картки, свайп для перегляду варіантів, параметри в bottom-sheet.

## Візуальна мова
- Теплі природні відтінки Ci Grass і Ci Rose; хвилі Ci Sky для легкості.
- Glow/SoftGlow лише як підсвітка активних варіантів.
- Чіткі бордери Ci Deep для відділення прев’ю.
