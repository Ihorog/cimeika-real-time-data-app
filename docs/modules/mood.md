# Настрій — емоційний шар

## Огляд
Настрій відображає поточний емоційний стан користувача через хвилі та карти настроїв, синхронізовані з Ci.

## Ключові екрани
- **Емоційні хвилі:** живі градієнти, що реагують на зміни показників.
- **Панель маркерів:** бейджі настрою, тригери рекомендацій та коротких порад.
- **Зрізи даних:** історія настрою з `/ci/data` для зв’язку з подіями.

## Основні компоненти
- Перемикач режимів «спокій / дія / творчість».
- Слайдер інтенсивності з кольорами Ci Calm та Ci Wave.
- Блок «порада від Ci» на базі поточного контексту.

## API взаємодія
- Читання даних настрою через `/ci/data`.
- Відправка сигналів у ПоДію та Календар для прогнозу навантаження.

## Адаптивність
- **Desktop XL:** Canvas/SVG хвилі з плавними анімаціями.
- **Laptop:** спрощені хвилі, дві колонки (графіка + поради).
- **Mobile:** статичні градієнти, акордеон з маркерами настрою.

## Візуальна мова
- Домінують Ci Calm і Ci Wave; Ci Rose додає м’які акценти.
- Glow використовується лише як легкий ореол навколо ключових маркерів.
- Темний фон не застосовується, щоб зберегти легкість шару.
