# Галерея — фотоархів

## Огляд
Галерея є центром збереження зображень, синхронізованим з Малею та подіями.

## Ключові екрани
- **Слайдер/категорії:** Сім’я, Настрої, Події, Особисте.
- **Перегляд з hover-preview:** на Desktop показує метадані й посилання на події.
- **Завантаження:** форма `/api/gallery/upload` з валідацією.

## Основні компоненти
- Фільтри категорій і тегів.
- Кнопка «відправити в Маля» для повторного використання образів.
- Статуси синхронізації з датасетом `ci_power`.

## API взаємодія
- `/api/gallery/upload` для завантажень, обмін метаданими з Ci.
- Інтеграція з `ci_power` для навчання та генерації варіантів у Малі.

## Адаптивність
- **Desktop XL:** горизонтальний слайдер і панель деталей справа.
- **Laptop:** дві колонки (сітка + деталі).
- **Mobile:** вертикальний список з повноекранним переглядом при натисканні.

## Візуальна мова
- Нейтральний фон Ci Sky/Ci Deep, акценти Ci Wave.
- Легкі ореоли Glow/SoftGlow для обраних фото; без жовтих суцільних заливок.
- Плашки статусів на Ci Deep/Ci Shadow з текстом високого контрасту.
