# ПоДія — планер подій

## Огляд
ПоДія управляє сценаріями майбутнього: створення, активація та завершення подій через timeline.

## Ключові екрани
- **Timeline/сітка:** події зі статусами «заплановано», «активно», «виконано».
- **Картка події:** статус, теги, прив’язка до Календаря та Настрою.
- **Прогрів майбутнього:** світлові акценти Ci Glow у вузлах, що стартують найближчим часом.

## Основні компоненти
- Фільтри за статусом та типами подій.
- Дії свайпом/hover: «позначити виконаною», «пересунути», «зв’язати з настроєм».
- Плашка API-синхронізації з `/api/calendar/events`.

## API взаємодія
- Читання/запис подій через `/api/calendar/events`.
- Push оновлень до Ci для синхронізації з іншими модулями.

## Адаптивність
- **Desktop XL:** горизонтальний timeline з масштабуванням zoom.
- **Laptop:** дві колонки (timeline + деталі події).
- **Mobile:** списковий режим з акордеоном деталей, свайп для зміни статусу.

## Візуальна мова
- Фон Ci Sky, активні лінії Ci Wave.
- Теплі підсвітки Ci Glow/SoftGlow лише як ореоли майбутніх вузлів.
- Текст і бордери на Ci Deep/Ci Shadow для контрасту.
