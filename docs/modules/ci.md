# Ci — ядро системи

## Огляд
Ci є центральним координатором Cimeika: керує сенсами, API та станами модулів. Інтерфейс будується як панель керування з інтерактивною мапою зв’язків.

## Ключові екрани
- **Мапа вузлів:** граф модулів з лініями резонансів; hover підсвічує активність Ci Wave, обводка Ci Glow як ореол.
- **API-консоль:** вкладки `ci/chat`, `ci/data`, `ci/components` з історією запитів та швидкими пресетами.
- **Панель статусів:** живі індикатори модулів з сигналами активності/помилок.

## Основні компоненти
- Контролер запитів (форма + JSON-view).
- Тригери "sync", "failsafe", "learner loop" з чіткими hover/disabled станами.
- Віджет "semantic resonance" для вибору виконавця.

## API взаємодія
- `/ci/chat` — діалогові звернення з контекстом.
- `/ci/data` — агрегація показників для Настрою, Казкара та історій.
- `/ci/components` — опис UI-компонентів та їх станів.

## Адаптивність
- **Desktop XL:** розширений граф зв’язків (SVG/Canvas) і багатоколонкова консоль.
- **Laptop:** дві колонки: граф + вкладки API.
- **Mobile:** таби вузлів, спрощений JSON-view, основні кнопки винесені в нижню панель.

## Візуальна мова
- Базовий фон Ci Sky/Ci Deep; акценти Ci Wave.
- Glow/SoftGlow лише як підсвітка активних вузлів.
- Контрастні тексти на Ci Deep/Ci Shadow для читабельності.
