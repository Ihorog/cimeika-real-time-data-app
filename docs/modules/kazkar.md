# Казкар — архів історій

## Огляд
Казкар зберігає історії, легенди та пам’ять системи. Інтерфейс темний з м’яким золотим світлом.

## Ключові екрани
- **Таймлайн історій:** події минулого з маркерами важливості.
- **Перегляд сувою:** розгортання обраної історії з голосовим відтворенням (ID `ci_kazkar_voice`).
- **Пошук/фільтр:** за темою, датою, резонансом.

## Основні компоненти
- Карти історій з коротким синопсисом.
- Кнопки «програти голосом», «зв’язати з подією», «додати тег настрою».
- Статусні індикатори синхронізації з `/ci/data`.

## API взаємодія
- Читання історичних зрізів через `/ci/data`.
- Синхронізація аудіо/тексту для ПоДії, Настрою та Галереї.

## Адаптивність
- **Desktop XL:** повноекранний таймлайн з маркерами; детальна панель праворуч.
- **Laptop:** дві колонки (список + деталі).
- **Mobile:** вертикальні картки з акордеоном деталей; кнопки жестів для відтворення.

## Візуальна мова
- Темний фон Ci Shadow/Ci Black; текст на Ci Deep.
- Підсвітки Ci Glow/SoftGlow як ореоли навколо важливих вузлів.
- Акценти Ci Wave для інтерактивних контролів.
