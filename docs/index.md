# Cimeika Wiki

Екосистема Cimeika поєднує сенсорику, штучний інтелект та модулі взаємодії. Ця документація збирає основні сценарії, API та інфраструктурні процеси для швидкого старту та підтримки.

## Складові
- **Backend/API** — шар FastAPI з маршрутизацією для модулів Ci та інтеграцією зовнішніх сервісів.
- **Frontend** — Next.js застосунок для візуалізації резонансів і керування подіями.
- **Orchestrator** — логіка пріоритизації задач і взаємодії між модулями.
- **Docs** — MkDocs сайт для технічної та операційної інформації.

## Ops
- Health-check: `/health` повертає статус сервісу.
- Логування: backend налаштований на INFO-рівень з таймстемпами.
